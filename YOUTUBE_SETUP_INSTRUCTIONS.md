# 📺 НАСТРОЙКА YOUTUBE API ДЛЯ YTCONT

## 🎯 Что нужно сделать

Для загрузки видео на YouTube из YTCont необходимо настроить OAuth 2.0 авторизацию через Google Cloud Console.

## 📋 Пошаговая инструкция

### 1. Создание проекта в Google Cloud Console
1. Перейдите на https://console.cloud.google.com
2. Нажмите "Create Project" (если не создан)
3. Введите название: "YTCont Video Uploader"
4. Нажмите "Create"

### 2. Включение YouTube Data API v3
1. В меню слева выберите "APIs & Services" → "Library"
2. В поиске введите "YouTube Data API v3"
3. Нажмите на результат, затем "Enable"
4. Дождитесь активации (несколько секунд)

### 3. Создание OAuth 2.0 Credentials
1. В меню слева "APIs & Services" → "Credentials"
2. Нажмите "Create Credentials" → "OAuth 2.0 Client IDs"
3. Выберите тип приложения: "Desktop App"
4. Введите имя: "YTCont Desktop Client"
5. Нажмите "Create"

### 4. Скачивание client_secret.json
1. После создания credentials нажмите на название клиента
2. Нажмите кнопку "Download JSON" в правом верхнем углу
3. Файл будет скачан как `client_secrets_<номер>.json`
4. Переименуйте файл в `client_secret.json`

### 5. Размещение файла
Поместите `client_secret.json` в корневую папку проекта YTCont (рядом с main.py)

## 🚀 Первая авторизация

После размещения файла:
1. Перезапустите YTCont (или main.py)
2. Перейдите в веб-интерфейс (обычно http://localhost:8000)
3. Нажмите кнопку "⚙️ Настроить YouTube API" в секции YouTube Upload
4. Проверьте статус - должна появиться форма загрузки
5. При первой попытке загрузки откроется браузер для авторизации
6. Войдите в аккаунт Google, привязанный к каналу YouTube
7. Разрешите доступ к YouTube каналу
8. Токен сохранится в `token.pickle` (не коммитьте в Git!)

## ⚠️ Важные моменты

### Безопасность
- **НЕ делитесь** `client_secret.json` и `token.pickle`
- Добавьте эти файлы в `.gitignore`:
  ```
  client_secret.json
  token.pickle
  youtube_token.pickle
  youtube_uploads.json
  ```

### Ограничения YouTube
1. **Максимальный размер видео:** 256GB
2. **Максимальная длина:** 12 часов
3. **Формат файла:** MP4, AVI, MOV, MKV, WebM
4. **Длина описания:** до 5000 символов
5. **Настройка публикации:**
   - `public` - для всех
   - `private` - только вы (по умолчанию при отложенной публикации)
   - `unlisted` - только по ссылке

### Отложенная публикация
1. Минимум 15 минут от текущего времени
2. Максимум 6 месяцев вперед
3. Формат времени: RFC 3339 (ISO 8601)
4. Пример: `2024-12-25T18:00:00Z` (25 декабря в 18:00 по UTC)

## 🔧 Решение проблем

### "Файл client_secret.json не найден"
- Проверьте что файл в корне проекта
- Проверьте название файла (должно быть `client_secret.json`)
- Попробуйте перезагрузить страницу

### "Ошибка авторизации"
1. Удалите файл `token.pickle`
2. Перезапустите приложение
3. Попробуйте авторизоваться снова

### "Missing required packages"
Установите недостающие пакеты:
```bash
pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib
```

### "Доступ запрещен"
- Проверьте что YouTube канал подтвержден
- На канале не должно быть ограничений
- Убедитесь что вы вошли в правильный аккаунт Google
